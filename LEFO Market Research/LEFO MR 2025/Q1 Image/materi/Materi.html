Augmentasi images

Pada setiap model image classifier, dikenal suatu teknik bernama image augmentation yang berguna untuk memperbanyak images sehingga model bisa lebih pintar menebak image yang diberikan.

Summary: input data

  1. Data yang digunakan: images.
    • Agar model semakin pintar, kita bisa memperbanyak images dengan teknik image augmentation.
  2. Images dilihat sebagai matriks dengan sifat:
    • Dimensi merupakan ukuran height dan width (pixel size).
    • Banyaknya layers tergantung dari format jpeg dan png yang mengakibatkan perbedaan ukuran matrix.
  3. Matriks ini kelak yang akan digunakan untuk membangun model deep learning.
    • Oleh karena itu ukuran matriks harus konsisten selama pembuatan model.

Proses Pembuatan Model Deep Learning

Deep Learning: Artificial Neural Network

Ini adalah arsitektur dari ANN. Cara kerjanya adalah regresi linear. Big regression machine.

Jelasin layers dan nodes

Apa hubungan arsitektur ANN dengan dengan ukuran matriks?

Input layers harus memuat nodes seukuran dengan matriks input.

Output layers harus memuat berapa banyak nodes?

Output layers tergantung dari output prediksinya apa?

Kalau kita hendak memprediksi 2 kategorik, misal: ayam atau kambing, ya outputnya hanya 2.

Bagaimana Langkah Kerjanya?

  1. Siapkan minimal 2 kategori images yang hendak digunakan.
    • Lakukan augmentasi.
  2. Pisahkan images menjadi dua:
    • Train datasets: model akan dilatih menggunakan kumpulan images ini.
    • Test datasets: model akan divalidasi menggunakan kumpulan images ini. Analogi: semacam ujian setelah seorang siswa belajar.
  3. Buat arsitektur ANN.
    • Perhatikan ukuran matriks dan output nodes-nya.
  4. Buat aturan (baca: definisikan) program wajib belajar.
    • Berapa banyak epochs?
    • Berapa learning rate?
    • Berapa porsi validasi saat belajar?
    • Berapa batch yang digunakan dalam satu epoch pembelajaran?
    • Activation function setiap layer mau seperti apa?
  5. Suruh modelnya belajar.
  6. Validasi modelnya dan lihat seberapa akurat modelnya mengenali images.
    • Jika belum pintar, lakukan wajib belajar lagi.

Ini bedanya DL dengan ML. DL gak harus ngulang saat ada data baru. kalau ML kan ngulang modelnya ya.

Ilustrasi Kerja dengan TensorFlow

Salah satu framework yang digunakan untuk membuat model deep learning adalah TensorFlow.

  • Sebuah platform atau library open-source yang dibuat oleh Google.
  • Digunakan untuk membangun dan melatih model Deep Learning.
  • Menyediakan alat, fungsi siap pakai, dan lingkungan untuk memudahkan proses pengembangan model AI.
  • Fleksibel: Bisa dijalankan di berbagai perangkat keras (CPU, GPU, atau TPU).

TensorFlow bukan model AI itu sendiri, tapi alat untuk membuat model tersebut.

Analogi: TensorFlow itu Seperti Dapur Profesional

Skenario: Anda ingin membuat kue yang sangat kompleks (Model Deep Learning).

  • Deep Learning: Adalah resep dan konsep cara membuat kue tersebut (prinsip jaringan saraf, layers, dll).
  • TensorFlow: Adalah dapur profesional yang lengkap:
    • Menyediakan mangkuk, mixer, oven canggih (fungsi matematika, algoritma optimasi, struktur data ‘Tensor’).
    • Menyediakan alat ukur presisi (tools debugging dan monitoring).
    • Menyediakan tata letak dapur yang efisien (framework yang terstruktur).
    • Memungkinkan Anda fokus meracik bahan (data) dan mengikuti resep (membangun model) tanpa perlu membuat semua alat dari nol.

TensorFlow menyederhanakan dan mempercepat proses “memasak” (membangun dan melatih) model Deep Learning.
